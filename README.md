# Atelier YO — Status

Public status page + uptime monitoring for **atelier-yo.fr** (powered by Upptime).

- **Status page:** https://status.atelier-yo.fr/
- **Incidents:** https://status.atelier-yo.fr/history
- **Main website:** https://atelier-yo.fr

## Monitored services

- https://atelier-yo.fr
- https://www.atelier-yo.fr

## How it works

- Checks run automatically via **GitHub Actions**
- History + response-time graphs are stored in this repository
- Incidents are tracked using **GitHub Issues**

## Workflows

- **Uptime CI**: performs checks and records up/down events
- **Response Time CI / Graphs CI**: generates response-time data and graphs
- **Deploy Custom Status Site**: publishes the website to GitHub Pages
